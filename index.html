Vision Statement

Our vision is to see people use our product, so they can properly recycle and educate themselves and others.

The Idea Behind CodeGreen

Every year 91% of plastic is not being properly recycled and it ends up harming the animals and habitats around us, but did you know it will also harm us? Based on the alarming data on pollution, we wanted to create an easy to use device to bring awareness to recycling.

Information

Due to unproperly disposing trash and recycalables, we have faced many issues and some issues we will face in the future if we do not start recycling properly.

Pollution

Due to improperly disposing trash and recyclables, we are polluting the Earth. We have caused landfills, water pollution, air pollution, and many more. While polluting the Earth, we are harming the land, the animals, and the human race.

Green House Gasses

These dangerous gasses are caused when the chemicals and trash enter the oceans which cause it to release huge amounts of carbon dioxide.

Global Warming

The unnecessary waste can end up in landfills and cause the harmful chemicals to seep into the soil or mix into large bodies of water; we end up destroying natural habitats and natural resources. This leads to green house gasses to be released and contributes to global warming.


Our Project

To make our easy to use device, we spent many hours working on scripts, trial and error, and several images. We have listed what we used in our project below:

Raspberry Pi Kit

The Raspberry Pi was used to run our model on the teachable machine.The kit includes:

Includes Raspberry Pi 4 Model B with 1.5GHz 64-bit quad-core CPU (2GB RAM)

32GB Samsung EVO+ Micro SD Card (Class 10) Pre-loaded with NOOBS, USB MicroSD Card Reader

CanaKit Premium High-Gloss Raspberry Pi 4 Case with Fan Mount, CanaKit Low Noise Bearing System Fan

CanaKit 3.5A USB-C Raspberry Pi 4 Power Supply with Noise Filter, Set of Heat Sinks, Micro HDMI to HDMI Cable - 6 foot (Supports up to 4K 60p)

CanaKit USB-C PiSwitch (On/Off Power Switch for Raspberry Pi 4)

Raspberry Pi Camera

We used the Raspberry Pi Camera to input the image of a product, so it could display the output on the monitor. We used a Arducam 5MP Camera for Raspberry Pi, 1080P HD OV5647 Camera Module V1 for Pi 4, Raspberry Pi 3, 3B+, and Other A/B Series.

TensorFlow

TensorFlow is an open-source software library that aids in machine learning and training. We used this to to aid the Raspberry Pi to help it detect things.

Teachable Machine

The teachable machine is used to input pictures or files in order to recognize the object. This was necessary because it created a trained model which we could import to the Raspberry Pi.
